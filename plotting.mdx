## <kbd>function</kbd> `plot_series`

```python
plot_series(
    df: Optional[~DFType] = None,
    forecasts_df: Optional[~DFType] = None,
    ids: Optional[List[str]] = None,
    plot_random: bool = True,
    max_ids: int = 8,
    models: Optional[List[str]] = None,
    level: Optional[List[float]] = None,
    max_insample_length: Optional[int] = None,
    plot_anomalies: bool = False,
    engine: str = 'matplotlib',
    palette: Optional[str] = None,
    id_col: str = 'unique_id',
    time_col: str = 'ds',
    target_col: str = 'y',
    seed: int = 0,
    resampler_kwargs: Optional[Dict] = None,
    ax: Optional[Axes, ndarray, ForwardRef('Figure')] = None
)
```

Plot forecasts and insample values. 



**Args:**
 
 - <b>`df`</b> (pandas or polars DataFrame, optional):  DataFrame with columns  [`id_col`, `time_col`, `target_col`]. Defaults to None. 
 - <b>`forecasts_df`</b> (pandas or polars DataFrame, optional):  DataFrame with  columns [`id_col`, `time_col`] and models. Defaults to None. 
 - <b>`ids`</b> (list of str, optional):  Time Series to plot. If None, time series  are selected randomly. Defaults to None. 
 - <b>`plot_random`</b> (bool, optional):  Select time series to plot randomly.  Defaults to True. 
 - <b>`max_ids`</b> (int, optional):  Maximum number of ids to plot. Defaults to 8. 
 - <b>`models`</b> (list of str, optional):  Models to plot. Defaults to None. 
 - <b>`level`</b> (list of float, optional):  Prediction intervals to plot.  Defaults to None. 
 - <b>`max_insample_length`</b> (int, optional):  Maximum number of train/insample  observations to be plotted. Defaults to None. 
 - <b>`plot_anomalies`</b> (bool, optional):  Plot anomalies for each prediction  interval. Defaults to False. 
 - <b>`engine`</b> (str, optional):  Library used to plot. 'plotly', 'plotly-resampler'  or 'matplotlib'. Defaults to 'matplotlib'. 
 - <b>`palette`</b> (str, optional):  Name of the matplotlib colormap to use for the  plots. If None, uses the current style. Defaults to None. 
 - <b>`id_col`</b> (str, optional):  Column that identifies each serie.  Defaults to 'unique_id'. 
 - <b>`time_col`</b> (str, optional):  Column that identifies each timestep, its  values can be timestamps or integers. Defaults to 'ds'. 
 - <b>`target_col`</b> (str, optional):  Column that contains the target.  Defaults to 'y'. 
 - <b>`seed`</b> (int, optional):  Seed used for the random number generator. Only  used if plot_random is True. Defaults to 0. 
 - <b>`resampler_kwargs`</b> (dict, optional):  Keyword arguments to be passed to  plotly-resampler constructor. For further custumization ("show_dash")  call the method, store the plotting object and add the extra arguments  to its `show_dash` method. Defaults to None. ax (matplotlib axes, array of matplotlib axes or plotly Figure, optional):  Object where plots will be added. Defaults to None. 



**Returns:**
 
 - <b>`matplotlib or plotly figure`</b>:  Plot's figure
