fill_gaps

 fill_gaps (df:pandas.core.frame.DataFrame, freq:Union[str,int],
            start:Union[str,int]='per_serie', end:Union[str,int]='global',
            id_col:str='unique_id', time_col:str='ds')

Enforce start and end datetimes for dataframe.

Type Default Details
df DataFrame Input data
freq typing.Union[str, int] Series‚Äô frequency
start typing.Union[str, int] per_serie Initial timestamp for the series.
* ‚Äòper_serie‚Äô uses each serie‚Äôs first timestamp

The default functionality is taking the current starts and only extending the end date to be the same for all series.

fill_gaps(
    df,
    freq='YS',
)
unique_id ds y
0 0 2020-01-01 0.0
1 0 2021-01-01 1.0
2 0 2022-01-01 NaN
3 0 2023-01-01 2.0
4 1 2021-01-01 3.0
5 1 2022-01-01 4.0
6 1 2023-01-01 NaN

We can also specify end='per_serie' to only fill possible gaps within each serie.

fill_gaps(
    df,
    freq='YS',
    end='per_serie',
)
unique_id ds y
0 0 2020-01-01 0.0
1 0 2021-01-01 1.0
2 0 2022-01-01 NaN
3 0 2023-01-01 2.0
4 1 2021-01-01 3.0
5 1 2022-01-01 4.0
